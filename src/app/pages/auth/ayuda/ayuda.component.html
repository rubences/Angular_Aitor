Para empezar a usar la página web, lo primero que hay que hacer es crear una cuenta, que se guardará en una base de datos.



Si ya se tiene una cuenta creada, solo habrá que introducir el correo electrónico y la contraseña para poder iniciar sesión.



Una vez el usuario este registrado, aparecerán las categorías disponibles.



Al seleccionar una de las categorías, aparecerán los productos disponibles.



Una vez seleccionado el producto, al pulsar el botón de comprar, nos lleva a la página donde está disponible el producto.



Para poder salir, habrá que pulsar cerrar sesión, donde nos trasladará a la página de inicio.
